# Student Group Distribution Optimizer
This is a copy version of the original.

## Project Overview
This project automates the distribution of 6,000 students into balanced teams based on specific academic criteria without external libraries.

## Key Technical Implementation
* **Language:** Pure Python 3.x
* **Data Structures:** Custom lists and dictionaries.
* **Performance:** Optimized logic to process 6,000+ data points efficiently.

## Algorithmic Approach
- **Parsing:** Custom file I/O for data handling.
- **Sorting:** Custom implementation of sorting logic for academic tiers.
- **Distribution:** Nested-loop assignment engine with diversity validation.
- **Calculation:** Custom functions for CGPA variance:
  Variance = Σ((x - μ)^2) / n

## Features
* 99.3% faculty representation diversity.
* Balanced CGPA variance (0.02/5.0).
* Adherence to zero-dependency constraints.
